# KOCOM RS485 프로토콜 명령어 명세 (롯데캐슬/코콤 월패드)

본 문서는 코콤 월패드 시스템에서 사용되는 RS485 제어 패킷의 모든 경우의 수를 분석하여 정리한 기술 문서입니다.

## 1. 패킷 구조 (Packet Structure)

| 필드명 | 크기 | 설명 |
| :--- | :--- | :--- |
| Header | 2 bytes | `AA 55` (고정) |
| Type | 3 bytes | `30 BC 00` (제어/상태 타입) |
| Dest Device | 1 byte | 대상 기기 타입 코드 |
| Dest Room | 1 byte | 대상 방 번호 (0x00: 거실, 0x01: 침실1...) |
| Src Device | 1 byte | 송신 기기 타입 (0x01: 월패드 고정) |
| Src Room | 1 byte | 송신 방 번호 (0x00 고정) |
| Command | 1 byte | 명령 구분 (0x00: 일반제어, 0x02: 상태조회 등) |
| Data | 8 bytes | 상세 제어 페이로드 (ON/OFF, 온도, 모드 등) |
| Checksum | 1 byte | Header 제외한 16바이트 합의 256 나머지 |
| Tail | 2 bytes | `0D 0D` (종단문자) |
